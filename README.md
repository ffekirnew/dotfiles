## Usage
1. Clone the repository to your home directory.
```bash
cd ~
git clone https://github.com/ffekirnew/dotfiles.git
```

2. Run the symlink script.
```bash
cd dotfiles
chmod +x symlink.sh
./symlink.sh
```
